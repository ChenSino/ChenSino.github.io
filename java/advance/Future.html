                                    # 1、Future的作用

                                    Future可以用来获取一个异步执行的结果，可以使用isDone方法检查异步任务是否完成，或者使用get阻塞住调用线程，直到计算完成返回结果，你也可以使用cancel方法停止任务的执行。
